# Online Quiz App (Spring Boot)

## Gereksinimler
- Java 17
- PostgreSQL

## Calistirma
1. PostgreSQL'de bir veritabani olusturun:
   - DB: quizapp
   - Kullanici: postgres
   - Sifre: postgres

2. Uygulamayi calistirin:
   - `./mvnw spring-boot:run`

3. Varsayilan hesaplar:
   - Admin: admin / admin123
   - Kullanici: user / user123

## Ozellikler
- Kayit / giris
- Quiz cozum, skor ve gecmis
- Liderlik tablosu
- Admin paneli: quiz ve soru yonetimi, kullanici sonuclari
- Sure sayaci ve basit grafik
